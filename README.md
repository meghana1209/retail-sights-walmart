# Retail Sights – Walmart-Style Intelligent Retail Platform

## Final Year Project (BE – Artificial Intelligence & Data Science, 2026)

Retail Sights is a full-stack, Walmart-inspired retail intelligence platform designed as a final-year engineering project. The system focuses on intelligent product discovery, analytics-driven insights, and scalable frontend architecture.

## Project Objectives
- Build a scalable e-commerce frontend using modern web technologies
- Implement intelligent product insights and analytics
- Design a clean, responsive UI inspired by real-world retail platforms
- Integrate authentication and database services
- Apply AI/analytics concepts learned during the degree

## Tech Stack
- React (Vite)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase (Auth & Database)
- Node.js

## Features
- Product listing & category browsing
- Search & filter functionality
- User authentication
- Responsive UI (mobile + desktop)
- Scalable component-based architecture
- Analytics-ready data model

## Project Status
Actively under development as a final-year project with planned enhancements including:
- Recommendation system
- Admin dashboard
- Sales analytics
- Inventory insights
